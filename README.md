# ZSXQ-AI-Explainer

【南半球聊财经】知识星球 AI 解释器 - 一个帮助理解知识星球内容的浏览器扩展

![界面](docs/readme.jpg)

## 功能特点

- 🎯 专注于知识星球平台内容解释
- 🤖 基于 DeepSeek AI 提供智能解析
- 💡 经济学专业视角的内容分析
- 🔍 右键快捷操作，便捷实用

## 使用方法

1. 安装扩展后，点击扩展图标配置 DeepSeek API Key
2. 在知识星球页面内容区域右键
3. 点击"解释内容"选项
4. 等待 AI 分析结果在右侧弹窗显示

## 安装方法

1. 下载项目代码
2. 打开 Chrome 扩展管理页面 (chrome://extensions/)
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

## 配置说明

1. 点击扩展图标打开配置面板
2. 输入您的 DeepSeek API Key
3. 点击保存即可使用
